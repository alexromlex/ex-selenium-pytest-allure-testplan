How to use? ▼

You can filter by:

  • id - @allure.id("22") => {"id": {"eq": "22"}}
  • name - test name (function name: test_*) => {"name": {"eq": "test_get_partners_negative"}}
  • labels - @allure.epic("Epic") => {"labels": {"epic": {"eq": "API"}}}
  • markers - @pytest.mark.smoke => {"markers": {"in": ["smoke"]}}

Filter supports operators: eq, neq - for str valuest equal or not equal. in, nin - condition with array witch is include value or not. exists - boolean condition.

Filter strategies:

  • {key:value, key:value, ...} - filtered result by one condition with multiple params
  • [{key:value}, {key:value, key:value, ...}, ...] - filtered result + filtered result +...

EXAPMLES:

  • Testplan: Smoke, Release_02, Regression and Epic=API, exclude test_get_partners_negative test:
    [ {"markers": {"in": ["Release_02", "smoke"]}}, {"labels": {"severity": {"in": ["blocker", "critical"]}}}, {"labels": {"epic": {"eq": "API"}}, "name": {"neq": "test_get_partners_negative"}} ]
